
# Доступ пользователя. 
#Система должна проверять возраст пользователя.
#Если пользователю 18 лет или больше — доступ разрешается и выводится соответствующее сообщение "Доступ разрешен".
#В противном случае доступ запрещается. Результат проверки должен сохраняться и возвращаться для дальнейшего использования.

def check_age(x):
    result = x>=18
    if result:
        print("Доступ разрешен")
    else:
        print("Доступ запрещен")
    return result

    
```

```

#Применение скидки
#Система должна рассчитывать итоговую стоимость покупки.
#Если сумма заказа превышает 100 единиц, применяется скидка 10%.
#В противном случае цена остаётся без изменений.
#Итоговая стоимость должна выводиться и возвращаться.

def total_price(y):
    if y > 100:
        result = y * 0.9
    else:
        result = y
    print("Итоговая цена", result)
```


```python
total_price(2500)
```

    Итоговая цена 2250.0
    


```python
total_price(50)
```

    Итоговая цена 50

#Аутентификация

#Рабочее время
#Система должна определять, находится ли текущее время в рабочем диапазоне.
#Рабочими считаются часы с 9 до 18 включительно.
#Если текущее время попадает в этот диапазон — выводится сообщение о рабочем времени, иначе — о нерабочем.
#Результат должен возвращаться.

def check_time_zone(x):
    result = x >= 9 and x <= 18
    if result:
        print("Рабочее время")
    else:

    return result

#В словарях price и bought даны цены на фрукты и количество купленных килограммов. Нужно посчитать, сколько денег потратили только на фрукты с ценой выше 150 рублей за кг.

price = {
    "яблоки": 120,
    "бананы": 110,
    "клубника": 350,
    "манго": 400,
    "апельсины": 130
}

bought = {
    "яблоки": 2,
    "бананы": 3,
    "клубника": 1,
    "манго": 0.5,
    "апельсины": 2.5
}

count_sum = 0

    if price[i] > 150:
        count_sum = count_sum + (price[i] * bought[i])
print(count_sum)

```python
#В словаре inventory хранится количество товаров на складе. В словаре price_per_item — цена одного товара. Нужно посчитать общую стоимость всех товаров, у которых количество на складе больше 10 штук, но при этом исключите товар "бракованные" (его не нужно учитывать, даже если его больше 10).

inventory = {
    "ручки": 45,
    "карандаши": 8,
    "тетради": 23,
    "линейки": 5,
    "бракованные": 50,
    "ластики": 12
}

price_per_item = {
    "ручки": 15,
    "карандаши": 10,
    "тетради": 35,
    "линейки": 20,
    "бракованные": 0,
    "ластики": 8
}

total_sum = 0
for item in inventory: 
    if item != "бракованные" and inventory[item] > 10:
        total_sum += inventory[item] * price_per_item[item]
print(total_sum)
```
